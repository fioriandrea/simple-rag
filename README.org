* Simple RAG

Simple script to create basic RAG using [[https://github.com/abetlen/llama-cpp-python][llama-cpp-python]] and chromadb for educational purposes.  Not suitable for production.

It uses naive / length-based chunking of the input data.

** Dependencies

Create an environment and install the project with test dependencies:

#+begin_src shell
python -m venv .venv
. .venv/bin/activate
python -m pip install '.[dev]'
#+end_src

For compilation of llama-cpp-python, refer to [[https://github.com/abetlen/llama-cpp-python][it's repo]].

** Tests

#+begin_src shell
python -m pytest
#+end_src

** Usage examples

Show help:

#+begin_src shell
simple-rag --help
#+end_src

To create a db from text files:

#+begin_src shell
simple-rag dbgen --db ./mydb --model-path ./embedding.gguf --glob './docs/*.txt'
simple-rag dbgen --db ./mydb --model-path ./embedding.gguf --files ./a.txt ./b.txt
simple-rag dbgen --db ./mydb --model-path ./embedding.gguf --files-from ./files.txt
#+end_src

To add files to an existing db, replacing them first if already indexed:

#+begin_src shell
simple-rag dbgen --db ./mydb --files ./newdoc.txt --append
#+end_src

To sync indexed files with disk:

#+begin_src shell
simple-rag dbgen --db ./mydb --sync-indexed
#+end_src

If dbgen is interrupted, resume it with:

#+begin_src shell
simple-rag dbgen --db ./mydb --resume
#+end_src

To query a created db:

#+begin_src shell
simple-rag query --db ./mydb --query 'notes about linux command line'
simple-rag query --db ./mydb --query 'notes about linux command line' --json
#+end_src

To delete indexed chunks for one or more files:

#+begin_src shell
simple-rag delete --db ./mydb --files ./docs/old.txt ./docs/other.txt
#+end_src

To launch MCP server:

#+begin_src shell
simple-rag mcp --db ./mydb --description 'Project documentation for my application'
simple-rag mcp --db ./mydb --description 'Project documentation for my application' --transport stdio
#+end_src
